The Problem

AI Development Shouldn't Be This Fragmented

😤

Siloed Environments

Agent harnesses like Claude Code, Cursor, and OpenCode have incompatible configs, losing your context between IDEs.

🧠

Amnesia

Agents forget previous plans, context, and decisions between sessions, forcing you to start over every time.

🛡️

Security Risks

Executing agent code directly without sandbox limits or sanitization exposes your environment to vulnerabilities.

ECC solves this.

ECC is a complete system that provides skills, instincts, memory optimization, and security scanning out of the box, working natively across all major AI agent harnesses.

How to Use

01

Add the Marketplace

Connect Claude Code to the ECC marketplace repository.

/plugin marketplace add https://github.com/affaan-m/ECC
02

Install the Plugin

Install the base ECC plugin into your current environment.

03

Execute Workflows

Use any of the 260+ available skills and commands to plan, build, and deploy features.

# Example usage in terminal
